4. Method Summary
General approach.The team proposed a multi-stage
pipeline for merging 9 raw frames organized in three ex-
posure groups (short, middle, long) to generate a single
high-quality HDR RGB image. The method consists of
three main components: (1) a multi-scale cross-attention
alignment[2] module for spatial alignment, (2) a multi-scale
gated module for adaptive feature combination[10205094],
and (3) a SplitterNet[1] for final HDR reconstruction.Preprocessing.Exposure compensation is performed to
normalize intensity ranges across different exposure groups
using the equationX i=Ii/ti, wheret irepresents expo-
sure time. Relative exposure values{−2,0,2}are used for
short, middle, and long exposure groups respectively due to
unavailability of absolute exposure value information. Pixel
unshuffle operation are performed on every raw data to split
the color channel to avoid cross color interference in intial
layer.

The proposed module performs adaptive feature selection
using learnable gating functions at multiple resolutions, en-

tures from each input.

Reconstruction.For the final HDR reconstruction, we
employ an architecture similar to SplitterNet that processes
and fuse all the feature maps. The network first performs
same exposure blending by grouping features by exposure
level (short, medium, long) and applies channel attention
followed by convolution blocks to fuse features within eachexposure group. The three fused exposure feature maps are
then combined and passed through an initial convolution to
produce the input to the SplitterNet encoder.
The encoder follows a hierarchical splitting pattern,
where feature maps are recursively split into two parts
along the channel dimension and processed by convolu-
tional blocks. This splitting strategy allows the network to
process information in a tree-like structure, increasing the
effective receptive field while maintaining computational
efficiency.
The bottleneck features consist of 16 MidBlocks, each
incorporating channel attention, spatial attention, and resid-
ual connections. These blocks enhance the feature represen-
tations by focusing on both important channels and spatial
regions, allowing the network to capture more discrimina-
tive features.
The decoder follows a reverse tree structure, where fea-
tures from sibling branches are concatenated and upsam-
pled using transposed convolutions. This design allows the
network to effectively combine multi-scale features while
preserving spatial details through skip connections. Finally,
a pixel shuffle operation with factor 2 is applied to increase
the resolution, followed by a sigmoid activation to produce
the final HDR RGB image in the range [0, 1].
5. Experiments and Results Summary
Dataset.The method was evaluated on the NTIRE 2026
Burst HDR and Restoration challenge dataset containing
RAW burst images with varying exposures and ground truth
RGB images. The training set has 300 scenes, each with 9
RAW input frames following Bayer patterns (GRBG) and
one ground truth RGB image of resolution768×1536pix-
els. The validation set comprises 20 scenes with the same
structure. The nine input frames are captured at three expo-
sure levels: reference frame (in-0) with short exposure and
highest noise, frames in-1 and in-2 with same short expo-
sure, frames in-3 to in-5 with medium exposure and mod-
erate noise, and frames in-6 to in-8 with long exposure and
lowest noise levels.
Implementation details.The model was trained exclu-
sively on the provided dataset with raw-aware random hor-
izontal/vertical flipping and random cropping to256×256
patches. Implementation used PyTorch framework with
Adam optimizer (β 1= 0.9,β 2= 0.999). The initial learn-
ing rate was8×10−4, reduced to5×10−6using Step Learn-
ing Rate Scheduler. Training used only PSNR loss function
on a single NVIDIA A6000 GPU for 10k epochs. For vali-
dation and test phases, we used full frame input (768x1536)
to the model.
Evaluation metrics.Performance was evaluated using
Peak Signal-to-Noise Ratio (PSNR) measuring reconstruc-
tion quality by comparing pixel values between generated
and ground truth images, and Structural Similarity Index
(SSIM) evaluating perceptual quality by measuring struc-
tural similarity between images with values in[0,1]range.
Quantitative results.As shown in Table 1, presents the
quantitative performance comparison between the concate-
nated U-net model, SOTA TMRNet and our proposed
method on the validation set. Our method having least
parameters and FLOPS outperforms concatenated U-net
model and has slightly less PSNR when compared to TM-
RNet.
Computational efficiency.The model is designed for on-
device deployment in mobile applications with carefully
controlled computational complexity and memory footprint
to ensure real-time processing capability, as demonstrated

6. Conclusion
The alignment-centric framework focuses on reducing
alignment errors while optimizing computational efficiency.
Unlike traditional fusion-driven approaches, the method
prioritizes accurate alignment to mitigate error accumula-
tion. PixelUnshuffle decreases spatial resolution while in-
creasing channel dimensionality to expand receptive field
and reduce computational cost. A lightweight fusion mod-
ule enables high-quality HDR reconstruction under limited
resource constraints. Extensive experiments demonstrate
competitive performance on the NTIRE 2026 dataset while
maintaining computational efficiency suitable for mobile
deployment.
